# 欢迎使用 YiziMarkdown

一款简洁精致的 Windows 便携 `Markdown` 编辑器。免安装，解压即用。

为什么要开发一款 `Markdown` 编辑器？`Typora` 很漂亮，但要收费，免费的各种 `Markdown` 工具，太复杂也太丑，始终找不到趁手的 `Markdown` 编辑器。

于是，就然我的龙虾，帮我开发了一个。

用起来感觉还不错，这个文档，也是我的虾帮我写的，我用 `YiziMarkdown` 改了改。

## 快速上手

打开文件：`Ctrl+O` / 新建：`Ctrl+N` / 保存：`Ctrl+S` / 搜索：`Ctrl+F` / 关闭标签：`Ctrl+W` / 深浅切换：`F2` / 视图循环：`F3` / 快捷键大全：`F1`

## 多文件 Tab

单实例模式：双击 .md 文件不会打开多个窗口，自动合并到已有实例。已打开的文件会自动定位到对应标签页。

顶部 Tab 栏管理多个打开的文件：

- **首页** — Tab 栏最左侧的「首页」标签，展示最近打开的文件列表，点击即可打开
- **文件标签** — 每个打开的文件对应一个 Tab，显示文件名，未保存的文件会有一个`会呼吸的小圆点`提示
- **新建** — 点击 Tab 栏 `+` 按钮新建空白文件
- **关闭** — 点击 Tab 上的 `x` 关闭文件

## 四种视图模式

Tab 栏右侧提供视图切换按钮：

- **源代码** — 纯代码编辑，专注写作
- **并排** — 左编辑右预览，实时对照
- **实时** — 所见即所得编辑，Markdown 标记自动隐藏
- **预览** — 纯渲染预览，沉浸阅读

实时模式下还支持 4 种动画效果（聚焦/闪光/辉光/涟漪），在 设置 → 实时模式 中选择。

切换视图时滚动位置自动同步，不会丢失阅读进度。

## 大纲

左侧侧栏默认展示文档大纲，自动提取全部标题层级。

- 滚动编辑区或预览区，大纲高亮跟随当前位置
- 点击大纲节点，编辑区和预览区同步跳转到对应标题
- 并排模式下，滚动任一侧，另一侧自动联动

## 数学公式

用 `$...$` 包裹行内公式（如 `$E=mc^2$`），用 `$$...$$` 独占一行渲染块级公式。工具栏 Σ 按钮可一键包裹选中文字。

## Mermaid 图表

用 Mermaid 语法描述图表，预览和并排模式下自动渲染为可视化流程图、时序图、甘特图等。工具栏图表按钮可插入模板。图表主题在 设置 → 插件 中切换。

## 表格

工具栏的表格按钮打开 8×8 行列网格，鼠标点选即可插入对应大小的表格，无需手写 Markdown 表格语法。

## 文件关联

设置 → 通用 → 文件关联，勾选「设为默认 Markdown 编辑器」后，双击系统中的 .md 文件会直接用 YiziMarkdown 打开。

取消勾选即恢复系统默认。

## 自定义与插件

| 文件 | 用途 |
|------|------|
| `user.css` | 自定义样式，覆盖在所有主题之后，优先级最高 |
| `keybindings.json` | 快捷键配置（也可在 设置 → 快捷键 中可视化编辑） |
| `themes/*.css` | 放入新主题文件，重启后在设置面板中可见 |
| `templates/*.md` | 文档模板，新建文件时可选择 |

内置插件（在 设置 → 插件 中管理）：
- **KaTeX** — LaTeX 数学公式渲染（行内 `$...$`，块级 `$$...$$`）
- **Mermaid** — 流程图、时序图、甘特图等图表渲染

如果想定制自己的主题，也很简单：
 1. 打开 `doubao` 或 `DeepSeek` 或其他的AI平台
 2. 随便从`themes`目录中，找个文件上传
 3. 告诉AI，你想改为什么风格，什么配色
 4. 把AI生成的css文件，保存到themes目录
 5. 重启一下程序就可以了。

## 其他设置

- 自动保存（很有必要，可以设定自动保存间隔时间）
- 预览排版的字体和行距，非常推荐小米字体`Misans`，好看

## 导出

支持导出为 **HTML**、**Markdown** 和 **纯文本** 三种格式，通过工具栏导出菜单操作。
